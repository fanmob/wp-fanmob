wp-fanmob
=========

Wordpress plugin for [FanMob](https://www.fanmob.us/).

Installation
------------

Download the [latest release](https://github.com/fanmob/wp-fanmob/releases) as a zip file and then follow the [instructions given at the Wordpress Codex][instructions].

Usage
-----

To embed a poll, use the `fanmob` shortcode:

```
[fanmob id="6dde9f02-5085-4b82-bcf3-1887e1d22229"]
```

This will embed the poll at <https://www.fanmob.us/p/6dde9f02-5085-4b82-bcf3-1887e1d22229/> in your post.

Requirements
------------

`wp-fanmob` requires PHP 5.3 or newer.

[instructions]: http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation
